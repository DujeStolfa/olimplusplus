Promjena sadržaja rječnika

dvije životne linije, za admina i za server

- admin je odabrao jezik i rječnik, to je implicirano

(ovo su sada sve slučajevi, to je ono opt i alt)

dodavanje
- admin šalje zahtjev za dodavanjem riječi u rječnik, server odgovara s listom rječi u tom jeziku
- admin odabire riječ koju želi dodati i šalje je serveru, server je stavlja u rječnik

brisanje
- admin šalje zahtjev za brisanjem riječi iz rječnika, server odgovara s listom riječi
- admin bira riječ za obrisati, šalje zahtjev, server odgovara s pitanjem koje traži potvrdu izbora
	(da ne obriše admin slučajno riječ or something)
- admin potvrdi, server briše iz rječnika
