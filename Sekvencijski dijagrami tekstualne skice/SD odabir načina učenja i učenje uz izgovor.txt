Učenje s izgovorom

3 životne linije, korisnik, server i API

- korisnik klinke gumb, server pali korisnikov mikrofon
- korisnik izgovara, server sluša
- korisnik stišće gumb opet, server završava snimanje
- server šalje API-ju audio i čeka odgovor
- kad server dobije odgovor, šalje rezultat korisniku
